# m-piscine

Annuaire de piscinistes généré automatiquement.

## Installation

```bash
npm install
npx prisma generate
npx prisma db push
```

## Développement

```bash
npm run dev
```

## Configuration

Copiez `.env.example` vers `.env.local` et configurez les variables d'environnement.

## Site généré par

Générateur de sites - b-photovoltaique.fr
